# Vanity::Ui

Measure user activity

## Usage

Visit `/vanity-ui` in a browser.

## Installation

Add this line to your application's Gemfile:


```ruby
gem "vanity-ui"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install vanity-ui
```

add `mount Vanity::Ui::Engine => "/vanity-ui"` to `config/routes.rb`.
